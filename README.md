# meta-init

Process subset of init file contents out-of-band.

## Usage

Unless you have a really good reason for using this, you
probably shouldn't.

### With file:

```
$ meta-init -meta-file /path/to/file
```

### With parts directory:

```
$ meta-init -meta-directory /path/to/directory
```
